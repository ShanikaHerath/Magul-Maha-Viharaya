# Magul Maha Viharaya Website

Official React website for Magul Maha Viharaya, a historic Buddhist temple in Sri Lanka. The project is built to preserve tradition while utilizing modern web technologies.

## 🌟 Features

- **Modern & Premium UI:** Beautiful glassmorphism design, vibrant color palettes, and elegant typography.
- **Fully Responsive:** Optimized for all devices, from large desktop monitors down to small 360px mobile screens.
- **Advanced Animations:** Smooth transitions, tactile button effects, and a morphing mobile hamburger menu.
- **Clean Architecture:** Component-based structure utilizing React Router for seamless navigation.

## 📖 Pages

- **Home:** Introduction, quick links, and temple highlights.
- **About:** History and vision of the temple.
- **Gallery:** High-quality image and video gallery.
- **Events:** Upcoming religious events and past galleries.
- **Contact:** Location details and an interactive contact form.
- **Donations:** Secure contribution options and information.

## 🛠️ Tech Stack

- **Framework:** [React](https://reactjs.org/) (Bootstrapped with Create React App)
- **Routing:** React Router
- **Styling:** Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid, Glassmorphism)

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository (if applicable).
2. Navigate to the project directory:
   ```bash
   cd Magul-Maha-Viharaya
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

To start the application in development mode:
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in your browser. The page will reload when you make changes.

### Building for Production

To build the app for production to the `build` folder:
```bash
npm run build
```
This correctly bundles React in production mode and optimizes the build for the best performance.
